What Happens When You Open a Website

  1. User opens a web browser like Chrome, Edge or Firefox.
  2. User enters a website address (URL) such as www.example.com.
  3. The browser contacts the DNS server to find the website IP address.
  4. DNS server returns the IP address of the requested website.
  5. The browser sends an HTTP request to the web server.
  6. The web server processes the request.
  7. The server sends website files such as HTML, CSS and JavaScript.
  8. The browser renders the files and displays the webpage.

Conclusion

Client-server architecture is a model where the client requests information and the server provides the response. When a user opens a website, the browser sends a request to the server and the server returns the webpage data which is displayed on the user's screen.
